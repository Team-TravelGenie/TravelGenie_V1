# 트립챗(TripChat)
> 사진을 보내주시면 비슷한 분위기의 여행지를 추천해 드려요!

사진을 업로드 하면 해당 사진에서 키워드를 추출해서 사용자에게 제공하고,
사용자가 선택한 키워드 기반으로 AI가 여행지를 추천해 줍니다.
