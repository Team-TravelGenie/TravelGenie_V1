# 트립챗(TripChat)
> 사진을 보내주시면 비슷한 분위기의 여행지를 추천해 드려요!

1️⃣ 좋았던 여행지 사진이나, SNS를 하다가 우연히 보고 저장한 여행지 사진을 보내주세요. AI가 사진에서 키워드를 찾아 드립니다. </br>
2️⃣ 원하는 키워드를 선택하면 해당 키워드를 기반으로 새로운 여행지를 추천해 드립니다. </br>
3️⃣ 추천된 여행지에 대해 더 궁금하다면 AI에게 질문해보세요! </br>
